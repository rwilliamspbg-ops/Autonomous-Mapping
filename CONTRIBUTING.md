# Contributing to Sovereign Map

## Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` from `.env.example` and add your `VITE_GEMINI_API_KEY`
4. Run `npm run dev`

## Code Standards
- Use TypeScript for all new logic.
- Modularize simulation logic in `simulators/`.
- Follow the cyberpunk aesthetic for UI components.
- Ensure all components are responsive.
